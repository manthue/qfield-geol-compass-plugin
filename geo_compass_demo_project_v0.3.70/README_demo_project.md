# Geo Compass Demo Project v0.3.70

Open `geo_compass_demo.qgs` in QField. The project-specific plugin file is `geo_compass_demo.qml`.

The project legend includes `geology_measurements`, sourced from `geology_measurements.geojson`. The layer uses separate planar and linear SVG symbols and starts with a non-rendered schema seed feature so QField recognizes the GeoJSON point layer before the first real measurement is saved.
